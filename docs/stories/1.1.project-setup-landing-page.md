# Story 1.1: Project Setup & Landing Page

## Status: Draft

## Story

**As a** visitor,
**I want** to see a professional landing page explaining Darts Community,
**so that** I understand the platform's value and can decide to sign up.

## Acceptance Criteria

1. Laravel 11+ project initialized with standard directory structure
2. Tailwind CSS loaded via CDN (no build process)
3. Landing page displays at root URL with:
   - Hero section with value proposition in French
   - Key features overview (profile, equipment, community)
   - Call-to-action buttons for registration/login
4. Page is mobile-responsive
5. Basic Blade layout template established for reuse
6. Git repository initialized and pushed to GitHub

## Tasks / Subtasks

- [ ] Task 1: Initialize Laravel 11+ project (AC: 1)
  - [ ] Run `composer create-project laravel/laravel darts-community`
  - [ ] Configure `.env` for SQLite development database
  - [ ] Verify Laravel installation works via `php artisan serve`

- [ ] Task 2: Configure Tailwind CSS via CDN (AC: 2)
  - [ ] Create `resources/views/layouts/guest.blade.php` layout
  - [ ] Add Tailwind Play CDN script in `<head>`
  - [ ] Add custom color configuration for brand (deep green, gold accents)

- [ ] Task 3: Create Landing Page (AC: 3, 4)
  - [ ] Create `resources/views/pages/home.blade.php`
  - [ ] Create `HomeController` with `index()` method
  - [ ] Define route `GET /` in `routes/web.php`

  - [ ] Implement Features section (3 cards: Profile, Equipment, Community)

**Tech Stack:**
- Laravel 11.x
- PHP 8.2+
- Tailwind CSS 3.x via CDN (Play CDN for MVP)
- SQLite for development

**Environment Variables (.env):**
```env
APP_NAME="Darts Community"
APP_URL=http://darts-community.test
DB_CONNECTION=sqlite
```

**Project Structure:**
```

├── app/
```
